= System Context

This is an _explanation_ page: it builds understanding of how *<<PROJECT_NAME>>* fits into its
environment. It corresponds to the https://c4model.com[C4 model]'s *Level 1 — System Context*:
the system as a single box, the people who use it, and the external systems it talks to.
